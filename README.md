# Gerador de Anagramas

Projeto simples com Angular no frontend e Spring Boot no backend.

## Tecnologias

Angular 19

Spring Boot 3.3.11

Java 17

Maven Wrapper

Visual Studio Code

## Como executar

### Backend com Spring Boot 3.3.11 (Java 17)  (porta 8080)

cd backend
mvn spring-boot:run


### frontend com Angular 19 (porta 4200)
cd frontend
ng serve
